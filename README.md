# NewCool Vivienda - SERVIU Chile

Portal educativo sobre vivienda y subsidios habitacionales en Chile.

## Demo

[https://vivienda.newcool.io](https://vivienda.newcool.io)

## Caracteristicas

- Buscador de oficinas SERVIU
- Guia de subsidios habitacionales
- Calculadora de financiamiento
- Proceso de postulacion
- Casa propia y arriendo

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
